1. git clone: adalah cara menyalin projek repositori

2. git init : adalah mengecek projek

3. git status : mengecek projek sudah terbaca atau tidak

4. git add . : adalah printah menyelek file dari folder

5. git commit -m "isi pesan" : adalah memberikan nama file push

6. git push : adalah ekspor projek ke internet

7. git pull : adalah jika projek mengulang ekspor ke internet

8. clear : adalah hapus untuk menggunakan git bash

9. cls : adalah hapuas untuk comment promt atau cmd
